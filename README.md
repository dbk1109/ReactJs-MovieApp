### React JS
- 바닐라에서는 먼저 만들고 JS로 가져와서 HTML을 수정하는 방식이라면 React JS는 우리가 해왔던 방식을 거꾸로 하고 있음.
- 인터렉티브한 웹을 만들기 위해 제작되어왔음.

### JSX
- HTML이랑 거의 같게 생겼기 때문에 이해하기도 쉽고 관리가 쉬움
- 대신 브라우저가 이해할 수 있도록 설치해야함. 
- babel을 통해 확인 가능함.
